Small plugin that displays a summary of your post types and taxonomies.

== Description ==

This plugin adds a page in admin which displays a summary of your post and term data.

== Installation ==

1. Download, unzip, and upload the plugin folder to `wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
